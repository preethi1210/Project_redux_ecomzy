# Ecomzy

Ecomzy is a React-based e-commerce frontend app for browsing products, managing a cart, and secure user authentication.

Features

Browse products by category or search
Add products to cart and checkout
User login and signup
Responsive design for all devices

Tech Stack

Frontend: React, Redux, React Router
Styling: CSS / TailwindCSS
API Requests: Axios / Fetch
Auth: JWT

Setup
git clone https://github.com/<your-username>/ecomzy-frontend.git
cd ecomzy-frontend
npm install
npm start


Set up .env with:

REACT_APP_API_URL=http://localhost:4000
REACT_APP_PAYMENT_KEY=your_payment_key
